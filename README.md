# Smart-Bookmark-App

<h2>About</h2>
<ul>
<li>It is a smart bookmark app which can allow users to sign up and login through google oauth account.</li>
<li>Logged in users can add or delete bookmarks to/from his/her app account.</li>
<li>Note:Users can see only their own bookmarks</li>
</ul>

<h2>Technologies Used</h2>
<ul>
<li>Next.js(App Router)</li>
<li>Supabase(Auth, Database, Realtime)</li>
<li>Tailwind CSS(For Styling) </li>
</ul>

<h2>Main Features</h2>
<ul>
<li>Users data can be refreshed automatically upon adding or deleting bookmarks.</li>
<li>Realtime updates</li>
<li>User Data privacy</li>
<li>Flexible to add or delete their own bookmarks</li>
</ul>
